Este proyecto tiene la finalidad de integrarse junto a un informe
lo que hice aqui fue transformar miles de datos para poder servilos luego en alguna parte para poder visualizarlos
como parte de aprendizaje de este proyecto aprendi varias cosas
1. python para analisis y dentro varias librerias pandas y matplotlib
2. uso de la terminal para usar python, durante la codificacion iba depurando con la terminal para ir viendo los datos y ir ajustando los prints para tener mejor lectura
3. utilizo varias herramientas para procesar datos y visualizar datos
4. se utiliza power bi para realizar un dashboard para visualizar datos

este proyecto tiene el fin de demostrar mis habilidades para realizar proceso ETL desde python, visualizar datos desde PowerBI y realizar un storytelling de los mismos
